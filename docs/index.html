What it does

Not a list generator. A decision guide.

🎯

Profiles you first

One-question-at-a-time interview covering your subjects, interests, grades, and constraints before any research begins.

🔍

Course research

Identifies which degrees match your A-level profile and confirmed interests — via live web search, not a hardcoded list.

🏛️

University comparison

Uses subject rankings, NSS scores, and graduate outcomes — not overall league tables, which measure research output not teaching quality.

🇪🇺

European options

Country-by-country breakdown including Germany (free tuition at most public unis), TU Delft, Bocconi, ETH Zurich, IE Madrid, and more.

📋

UCAS shortlist

Builds a realistic 5-choice shortlist — 1 reach, 2 targets, 1 insurance — with honest grade reality-checking, not flattery.

✍️

Personal statement

Guides you through the 2026 three-question UCAS format with structure, story arc coaching, and feedback on drafts.

It's honest, not encouraging

  • Never builds a shortlist without a grade anchor — "assume I'll do well" is not a grade
  • Never cites overall university rankings as evidence for a course — they measure research output, not teaching
  • Never quotes fees from memory — always web searches current figures before giving a number
  • Flags European options systematically, not just when asked
Getting started

Two ways to use it

Claude.ai · Any plan

Claude.ai Project

Works on any Claude plan, including free. State is tracked via a Profile Summary block you paste between sessions.

  1. Go to claude.ai → Projects → New project
  2. Click Edit project instructions
  3. Paste the contents of claude-ai-project-instructions.md
  4. Save — advisor is ready
Claude Code · Full features

Claude Code Skill

Full file-based progress tracking, PDF research briefs, and persistent session state. Requires Claude Code.

mkdir -p ~/.claude/skills/uni-advisor
BASE="https://raw.githubusercontent.com/PatrickCroft-Baker/uni-advisor/main/uni-advisor"
curl -o ~/.claude/skills/uni-advisor/SKILL.md "$BASE/SKILL.md"
curl -o ~/.claude/skills/uni-advisor/reference-template.md "$BASE/reference-template.md"
curl -o ~/.claude/skills/uni-advisor/generate-uni-pdf.py "$BASE/generate-uni-pdf.py"
curl -o ~/.claude/skills/uni-advisor/setup.sh "$BASE/setup.sh"
bash ~/.claude/skills/uni-advisor/setup.sh

Then type /uni-advisor to start.

Scope

Built for UK students

Built around the UK application system — UCAS, A-levels, UK universities, and European alternatives for UK applicants. If you're outside the UK you'll find gaps in the process guidance. European university coverage is included throughout.
